# stock-analyzer: 个股深度分析 Skill

## 触发方式

用户输入包含"分析"/"看一下"/"怎么样"/"如何" + 公司名/股票代码时自动激活。例如：
- "分析比亚迪"
- "看一下宁德时代"
- "茅台怎么样"
- "002594.SZ"

**区分**：选股/扫股/排行类走 `smart-stock-picker` skill；单只个股深度分析走本 skill。

---

## 可用功能

输入公司名称或代码，自动完成全流程分析：

### `/分析 <公司名或代码>`

一次性完成：符号查找 → 数据拉取 → 因子评分 → 五策略回测 → 投资建议

支持中文名模糊匹配（如"比亚迪"→002594.SZ），也支持直接输入代码。

---

## 执行流程

```bash
cd /root/workspace/stock/backend && source venv/bin/activate && python scripts/analyze_stock.py "<公司名或代码>"
```

如需跳过数据拉取（数据库已有数据时）：
```bash
python scripts/analyze_stock.py "<公司名或代码>" --no-fetch
```

如需 JSON 输出（供程序消费）：
```bash
python scripts/analyze_stock.py "<公司名或代码>" --json
```

### 脚本内部步骤

1. **符号解析** — 中文名→代码映射（内置常见股票表 + DB模糊搜索）
2. **标的注册** — 若 `symbol_info` 表中不存在则自动添加，状态设为 `active`
3. **K线拉取** — 新浪财经 API（备选东方财富），默认拉取近8年日线，增量更新
4. **因子计算** — RSI / MA排列 / KDJ / MACD / CMF / 布林带位置 / 金叉死叉
5. **五策略回测** — MA交叉 / MACD / KDJ / 布林带 / 网格，输出收益+夏普+胜率+交易列表
6. **风格判定** — 趋势型 vs 震荡型 vs 混合型
7. **投资建议** — 推荐策略 + 入场/出场条件 + 当前信号状态

## 技术约束

- 数据源：新浪财经（K线）+ 东方财富 Skills（基本面，月300次）
- 回测引擎：`app.services.backtest.run_backtest()`
- 首次分析某标的会自动拉取K线数据，耗时 3-10 秒
- 数据库已有数据则秒出结果
- 最少需要 60 条日线数据才能分析
- 脚本位置：`backend/scripts/analyze_stock.py`

---

## 已支持股票名称映射

内置 30+ 常见A股名称→代码映射，未覆盖的会通过数据库模糊搜索匹配。可通过编辑 `scripts/analyze_stock.py` 中的 `NAME_TO_SYMBOL` 字典扩展。
