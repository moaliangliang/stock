# mx-data: 妙想金融数据 Skill

## 触发方式

用户查询股票行情、财务数据、分红、股东信息等具体数据时激活。

## 可用功能

基于东方财富权威数据库，支持自然语言查询：

1. **行情类** — 股票、行业、板块、指数的实时行情、主力资金流向、估值
2. **财务类** — 上市公司基本信息、财务指标、高管、股东结构、融资数据
3. **关系经营类** — 股票、股东及高管之间的关联关系

## 使用方式

```bash
cd ~/skills/mx-data && python3 mx_data.py "<自然语言查询>"
```

示例：
```bash
python3 mx_data.py "贵州茅台今日收盘价 涨跌幅"
python3 mx_data.py "宁德时代近三年净利润 营业收入"
python3 mx_data.py "比亚迪公司简介 主营业务 成立时间"
```

## 输出

自动保存到 `~/mx_data/output/`：Excel 多 sheet 数据、原始 JSON、查询描述。

## 要求

- 环境变量 `MX_APIKEY` 已配置
- 网络可访问 `mkapi2.dfcfs.com`
